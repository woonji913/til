##### HTML

1. 클릭하면 https://www.ssafy.com/ 로 이동하는 버튼을 만드시오.

   ```html
   <a href="https://edu.ssafy.com/"><button>싸피</button></a>
   ```



2. 다음 태그에서 잘못된 부분을 찾으시오.

   ```html
   href > src
   ```

   ```html
   <a href="../image/my_photo.png">내사진</a>
   ```
